## 介绍
凯利公式（Kelly Criterion）是一种**用于资金管理和下注比例优化的数学公式**，最初由美国科学家约翰·凯利在1956年提出，广泛应用于**投资、博彩、量化交易**等领域。

---

## 📌 一、凯利公式的核心表达式

\\( f^* = \frac{bp - q}{b} \\)


其中：

- ( f^* )：**最优投注比例**（占总资金的比例）
    
- ( b )：赔率（净收益倍数，比如赢1赔2则 ( b=2 )）
    
- ( p )：获胜概率
    
- ( q = 1 - p )：失败概率
    

b: **净收益倍数（net odds）** 也就是你每投入1单位资金，纯赚多少

---
## 📊 二、直观理解

凯利公式回答一个问题：

👉 **在一个有优势的赌局或投资中，每次应该拿出多少资金下注，才能长期让资金增长最快？**

- 如果你有**正优势（p 足够大）** → 应该下注
    
- 如果没有优势 → 不下注
    
- 优势越大 → 下得越多（但仍有限制）
    

---

## 🧠 三、简单例子

假设：

- 胜率 ( p = 0.6 )
    
- 赔率 ( b = 1 )（赢了赚1倍）
    

代入公式：  
[  
f^* = \frac{1 \times 0.6 - 0.4}{1} = 0.2  
]

👉 结论：**每次下注 20% 的资金最优**

---

## ⚖️ 四、凯利公式的特点

### ✔ 优点

- 最大化**长期资金增长率**
    
- 自动控制风险（不会全仓）
    
- 理论上最优策略（在概率已知前提下）
    

### ⚠️ 缺点

- 对概率 ( p ) **非常敏感**（估错就会亏）
    
- 实际波动较大（可能短期大幅回撤）
    
- 不适合风险承受能力低的人
    

---

## 🧩 五、常见改进：分数凯利（Fractional Kelly）

实际中很多人不会用满凯利，而是：

- **0.5凯利（半凯利）**
    
- **0.25凯利（四分之一凯利）**
    

👉 原因：降低波动和回撤风险

---

## 🧭 六、应用场景

- 🎲 博彩（体育博彩、扑克）
    
- 📈 股票/期权交易
    
- 💰 投资组合资金分配
    
- 🤖 量化交易策略
